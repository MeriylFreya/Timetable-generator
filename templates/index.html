Welcome to College Timetable Generator

This system helps you automatically generate clash-free timetables for all classes with drag-and-drop editing capabilities.

Quick Start Guide:
  1. Add Teachers to the system
  2. Create Subjects and assign them to teachers
  3. Define Classes (e.g., CSE-A, ECE-B)
  4. Set Subject Requirements (how many periods per week for each subject in each class)
  5. Click Generate Timetable to auto-create schedules
  6. Use the Drag & Drop Editor to manually adjust the timetable
